- `15:45–16:10` JSBT: A Binary Serialization Format for Real JavaScript Object Graphs — **Alexander Cheprasov** (Senior SE · Valstro (UK))
  > Most serialization formats used in JavaScript were never designed for JavaScript. JSON can’t handle circular references or shared objects; Protobuf and Avro require schemas; MessagePack and CBOR still see data as a tree. In this talk, I will introduce JSBT – JavaScript Binary Transfer, a binary format designed specifically for JavaScript object graphs.
