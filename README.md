# CityJS London 2026 — Day 3

Mobile-first schedule viewer for CityJS London 2026 Day 3 (Fri 17 April).

- 4-track grid on desktop, vertical timeline on mobile
- Shortlist talks (★) — persisted in localStorage
- Filter by track or "My Picks"
- Dark mode via system preference

Plain static HTML — no build step.
